# audio.deletePlaylist

Delete a playlist from the audios for a specified user or community

Parameters:

|Name|Value|
|---|---|
|owner_id|User or community id (for example, obtained by users.get)|
|playlist_id|Playlist id|

Example (Kate):

```php
//tmpl-kate-common-start
$ownerId = 1;
$playlistId = 1;

curl_setopt(
    $ch,
    CURLOPT_URL,
    "https://api.vk.com/method/audio.deletePlaylist?access_token=".TOKEN.
    "&owner_id=$ownerId&playlist_id=$playlistId&v=5.95"
);
//tmpl-kate-common-end
```
