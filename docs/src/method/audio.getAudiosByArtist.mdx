# audio.getAudiosByArtist

Get audios by artist id

Parameters:

|Name|Value|
|---|---|
|artist_id|Artist id (for example returned by audio.searchArtists)|
|count (optional)|Maximum number of audios to return|
|offset (optional)|Offset to skip that number of audios|

Example (Kate):
```php
//tmpl-kate-common-start
$artistId = "1204512717686522332";

curl_setopt(
    $ch,
    CURLOPT_URL,
    "https://api.vk.com/method/audio.getAudiosByArtist?access_token=".TOKEN."&artist_id=".
    $artistId."&count=2&v=5.95"
);
//tmpl-kate-common-end
```
