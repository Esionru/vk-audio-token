# audio.delete

Delete an audio from audios for a specified user or community

Parameters:

|Name|Value|
|---|---|
|owner_id|User or community id (for example, obtained by users.get)|
|audio_id|Audio id|

Example (Kate):

```php
//tmpl-kate-common-start
$ownerId = 1;
$audioId = 1;

curl_setopt(
    $ch,
    CURLOPT_URL,
    "https://api.vk.com/method/audio.delete?access_token=".TOKEN."&owner_id=$ownerId&audio_id=$audioId&v=5.95"
);
//tmpl-kate-common-end
```
